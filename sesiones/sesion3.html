Sesión 3: Dispositivos de Red Básicos

Inicio Sesión 3

Dispositivos de Red Básicos

Objetivos de Aprendizaje

  • Identificar los diferentes dispositivos de red básicos
  • Comprender las funciones de switches y routers
  • Configurar dispositivos básicos en Packet Tracer
  • Establecer conectividad entre dispositivos

Dispositivos de Red Comunes

1. Hubs

  • Dispositivo de capa 1 (física)
  • Envía datos a todos los puertos
  • No tiene capacidad de filtrado
  • Poco utilizado en redes modernas

2. Switches

  • Dispositivo de capa 2 (enlace de datos)
  • Filtra y reenvía tramas basado en direcciones MAC
  • Crea dominios de colisión separados
  • Puede tener capacidades de capa 3 (multicapa)

3. Routers

  • Dispositivo de capa 3 (red)
  • Conecta diferentes redes
  • Toma decisiones de enrutamiento basadas en direcciones IP
  • Puede proporcionar seguridad básica

4. Puntos de Acceso Inalámbrico (WAP)

  • Permite la conexión de dispositivos inalámbricos
  • Conecta redes cableadas e inalámbricas
  • Puede incluir funciones de router (router inalámbrico)

Práctica: Configuración Básica de Dispositivos

Ejercicio 1: Configuración Básica de un Switch

En este ejercicio, configurarás un switch básico en Packet Tracer.

  1. Arrastra un switch 2960 al área de trabajo.
  2. Haz clic en el switch y ve a la pestaña "CLI".
  3. Ingresa los siguientes comandos: 
    enable
configure terminal
hostname MiSwitch
enable secret clase
line console 0
password cisco
login
exit
line vty 0 15
password cisco
login
exit
service password-encryption
ip default-gateway 192.168.1.1
interface vlan 1
ip address 192.168.1.10 255.255.255.0
no shutdown
end
copy running-config startup-config
  • enable: Entra en modo privilegiado
  • configure terminal: Entra en modo configuración global
  • hostname: Establece el nombre del dispositivo
  • enable secret: Establece la contraseña de modo privilegiado (encriptada)
  • line console 0: Configura la consola
  • line vty 0 15: Configura el acceso remoto (Telnet/SSH)
  • service password-encryption: Encripta las contraseñas
  • ip default-gateway: Establece la puerta de enlace predeterminada
  • interface vlan 1: Configura la interfaz de administración
  • copy running-config startup-config: Guarda la configuración

Ejercicio 2: Configuración Básica de un Router

Ahora configurarás un router básico y lo conectarás al switch.

  1. Arrastra un router 2911 al área de trabajo.
  2. Conecta el router al switch usando un cable de consola.
  3. Configura una dirección IP en la interfaz del router: 
    enable
configure terminal
interface GigabitEthernet0/0
ip address 192.168.1.1 255.255.255.0
no shutdown
exit
ip dhcp pool RED_LAN
network 192.168.1.0 255.255.255.0
default-router 192.168.1.1
dns-server 8.8.8.8
exit
ip dhcp excluded-address 192.168.1.1 192.168.1.10
end
copy running-config startup-config

Para verificar la configuración del router:

show ip interface brief  # Muestra el estado de las interfaces
show running-config    # Muestra la configuración actual
show ip route          # Muestra la tabla de enrutamiento

Ejercicio 3: Conectividad Básica

Verifica la conectividad entre dispositivos.

  1. Conecta dos PCs al switch.
  2. Configura las PCs para obtener dirección IP automáticamente (DHCP).
  3. Desde una PC, intenta hacer ping a la otra PC y al router.
  4. Verifica la conectividad a internet (si está configurada).

Si el ping falla, verifica:

  1. Que los cables estén correctamente conectados (deben estar en verde)
  2. Que las interfaces estén activas (no shutdown)
  3. Que las direcciones IP estén en la misma red
  4. Que no haya reglas de firewall bloqueando el tráfico
  5. Usa show ip interface brief para verificar el estado de las interfaces

Resumen de la Sesión

En esta sesión has aprendido sobre los dispositivos de red básicos, sus funciones y cómo configurarlos en Packet Tracer. Has practicado la configuración básica de switches y routers, y has verificado la conectividad entre dispositivos.
